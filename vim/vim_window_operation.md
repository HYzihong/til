# 窗口管理操作

### 新建窗口

#### 向右新建一个窗口

> `command + \` vscode 自带的

vim 自带的是 `C-w + v`

#### 向下新建窗口

改键
在 `C-p` 中输入 `>split editor up`> 进入设置> 改成 `C + command + /`

> `C + command + /`

vim 自带的是 `C-w + s`

### 删除窗口

#### 关闭单个窗口

> `command w` vscode 自带的

vim 自带的是 `C-w + c`

#### 清空全部窗口

> `command + k + w` vscode 自带的

#### 关闭其他窗口（包括侧边栏），保留当前窗口

> `C-w + o` vim

### 窗口选择

`command + 0~9`
command + 0 切换到侧边栏 空格键进入问津

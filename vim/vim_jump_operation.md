# 跳转操作

### 标记

单文件之间的跳转的时候所用标记 `m+小写字母` 一般为`mm`
多文件之间的跳转的时候所用标记 `m+大写字母` 一般为`mM`

> 可以记录多个标记

### 跳转到定义

`gd`

`j` `k` 可以上下选择
`h` `l` 可以实现折叠展开的操作

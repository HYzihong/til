# 插入操作

- 行首插入 `I`
- 行尾插入 `A` (包含空格)
- 行前插入 `O`
- 行后插入 `o`

> 特殊案例：
> 在`abcdefg `的 g 后面且在空格之前插入就需要`(Shift+l)L+i`

- 删除之后插入`c+动作（范围）`
